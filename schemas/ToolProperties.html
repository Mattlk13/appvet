Tool Properties XML Schema

The Tool Properties XML Schema defines the configurable parameters of a tool service. This schema is defined by the file ToolProperties.xsd. An example instance of this schema is shown in ToolProperties.xml.

Configurable instances of this schema are referred should be placed in $<TOOL_FILES_HOME>/conf/ToolProperties.xml. In general, tool services are developed and provided by vendors and users will not have direct access to these services.
