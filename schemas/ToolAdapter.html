Testing Service Adapter XML Schema

The Tool Adapter XML Schema defines the configurable parameters tool services to be integrated with AppVet. This schema is defined by the file ToolAdapter.xsd. An example instance of this schema is shown in ToolAdapter.xml.

Integration of a testing service adapter involves adding a vendor-supplied ToolAdapter.xml file into $APPVET_FILES_HOME/conf/tool_adapters by the AppVet administrator. In general, testing service adapters will be developed and provided by the vendors of the specific tool services and should not be modified by the AppVet administrator.
