Tool Adapter XML Schema

The Tool Adapter XML Schema defines the configurable parameters tool services to be integrated with AppVet. This schema is defined by the file ToolAdapter.xsd. An example instance of this schema is shown in ToolAdapter.xml.

Configurable instances of this schema are referred to as tool adapters> and should be placed in $APPVET_FILES_HOME/conf/tool_adapters. In general, tool adapters will be developed and provided by the vendor of the specific tool service.
