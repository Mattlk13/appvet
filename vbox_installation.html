CentOS VirtualBox Installation

  • Step 1: Install Oracle VirtualBox
  • Step 2: Install CentOS Image
  • Step 3: Import CentOS OVA
  • Step 4: Create CentOS Account (Optional)
  • Step 5: Install VirtualBox Guest Additions

Step 1: Install Oracle VirtualBox

Download and install Oracle VirtualBox 5.0.2 for your host environment from:

https://www.virtualbox.org/wiki/Downloads

Step 2. Install CentOS Image

Download the CentOS 7 Gnome torrent file from:

http://virtualboxes.org/images/centos

Download and install a torrent client and use the CentOS 7 Gnome torrent file to download the CentOS 7 Gnome Open Virtual Format (.ova) file. These directions also apply to a RedHat installation of AppVet.

Step 3. Import CentOS OVA

Start VirtualBox and select File > Import Appliance... to import the CentOS7-Gnome.ova file. Start the CentOS appliance by selecting Start from the VirtualBox toolbar. When the CentOS desktop appears, select CentOS as the username and use password reverse. Note that the root password for this installation is also reverse.

Step 4. Create CentOS Account (Optional)

Add an administrator user account (e.g., "John Doe") with username jdoe and password Jdoe123?. This account will be used to install all AppVet components (i.e., where $HOME=/home/jdoe).

Step 5. Install VirtualBox Guest Additions

After installing CentOS, install the VirtualBox Guest Additions. First, install the correct kernel-devel and kernel-headers so that they match the CentOS7 kernel. Open a terminal and run: 
		
	# sudo yum update
	# sudo yum install kernel-headers kernel-devel
	# sudo yum install tar bzip2 make automake gcc gcc-c++ kernel-devel
