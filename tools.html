Adding Tools

AppVet requires online tools to analyze and test uploaded apps. To add an existing online tool to AppVet involves placing a tool adapter into your $APPVET_FILES_HOME/conf/tool_adapters/<os> directory. A tool adapter is an XML file provided by the vendor (or developer) of the online tool that tells AppVet where and how to connect to the online tool. Note that tool adapters added to AppVet must correspond to an online tool that is available and accessible over the network.

Installing Tools

In some cases, an online tool will not be available or accessible over the network but will be available for download and installation on your local machine.
