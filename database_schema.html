Android Tool Status

The androidtoolstatus table defines the current processing or risk status of an Android tool for a given app. Table 7 shows the androidtoolstatus table schema. Note that AppVet comes with three internal tools: registration, appinfo, and audit. As new tool adapters are added to AppVet, AppVet automatically creates a new column in the androidtoolstatus table.

Table 8 defines the status table field descriptions.


Table 8. androidtoolstatus table fields.

Field Description
appid The AppVet generated unique ID.
registration The registration status of the app. The registration tool is a built-in internal tool for AppVet.
appinfo The Android pre-processing status. The appinfo tool is a built-in internal tool for AppVet.
audit The organization's audit status. The audit tool is a built-in internal tool for AppVet. Note that the status of an audit will override the current app status.
tool 1 The status of tool 1.
tool 2 The status of tool 2.
