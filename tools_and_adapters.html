Available Tools

The following tools are available for download. For more information, see AppVet Tools.

Android

  • Ad-Hoc Tool Adapter: A customizable adapter for tools not directly accessible by AppVet.
  • Androwarn: An AppVet service wrapper and tool adapter for Androwarn.
  • APK Signature Verifier: An AppVet service wrapper and tool adapter for the Oracle jarsigner to verify the signatures and integrity of Android APK files.
  • MasterKey-ExtraField Tester: A tool that tests for the presence of the Android MasterKey and ExtraField vulnerabilities.

iOS

Not available.
