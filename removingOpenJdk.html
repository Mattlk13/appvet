Removing OpenJDK

To view a list of preinstalled OpenJDK packages of Java, run:

# rpm -aq | grep -i java



To uninstall and remove OpenJDK Java packages, open a console and run:

# sudo rpm -e <package_name>



for each OpenJDK Java package. Note that other applications (e.g., libreoffice) might be dependent on OpenJDK and should be uninstalled (e.g., using yum remove libraoffice* ) prior to uninstalling OpenJDK.
