AppVet Properties

Configuration of AppVet is achieved via the AppVetProperties.xml file. This file is located in $APPVET_FILES_HOME/conf/AppVetProperties.xml . The AppVetProperties.xml file defines a number of configurable properties for AppVet. In this section, we describe the properties that may be configured for AppVet.

Admin

The Admin property allows for the specification of the AppVet administrator username and password. Setting Admin is required to launch AppVet for the first time.

Host

The Host parameter specifies the hostname (or IP address) of the AppVet server that client should use. Due to various issues with DNS servers in obtaining a correct hostname, AppVet requires the manual specification of this parameter. In addition, the port and SSL parameters for the server must also be specified.

Logging

The Logging parameter specifies the logging configuration of AppVet. The Level parameter specifies the types of logging statements to be output including DEBUG, INFO, WARN, and ERROR statements. In addition, the ToConsole parameter indicates whether logging should be output to the console, and is typically used for debugging during AppVet development.

UseSSO

The UseSSO parameter specifies whether AppVet should use Single Sign-On (SSO) for all transactions. If SSO is used, a separate URL that supports SSO will be used for AppVet to provide authentication.

Apps

The AppVet binary release requires the installation of platform dependencies. See instructions for installing AppVet dependencies.

Example

After AppVet dependencies have been installed, download the AppVet binary release Web Application Resource (WAR) file. After downloading the AppVet binary release, extract the compressed file onto your local machine.

AppVetProperties XML Schema

The AppVet binary release contains the following directories:
  • appvet.war: The AppVet web application.
  • appvet.images: AppVet images.
  • appvet.files: AppVet configuration files.

Tool Adapters

To install AppVet, copy the appvet.war and appvet.images files to your $CATALINA_HOME/webapps directory. Next, copy the appvet.files directory to your local machine.
